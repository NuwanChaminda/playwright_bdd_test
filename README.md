
# Playwright BDD Automation Framework

Production-ready QA automation framework using:

- Playwright (Python)
- Pytest
- Pytest-BDD
- Allure Reports
- Parallel execution
- Retry on failure
- Screenshot + video capture
- GitHub CI pipeline

## Install

pip install -r requirements.txt
playwright install

## Run tests

pytest

## Allure report

allure serve reports/allure-results
